A：我对自己的本科专业实在不感兴趣，政治学与行政学。机缘巧合下，我接触到插画，了解得越多越发现这个专业很适合我，于是，大三的时候我就开始准备作品集申请国外的学校。准备作品集的过程太快乐了，那时开始，我就想要成为一名插画师了。 Q：作为插画师，最令你兴奋的时刻是什么？ A：我是那种比较平静的人，不太会有大的情绪波动。我的喜悦感可能是来源于生活的，比如最近这几天参加了很有趣的workshop，印版画、做扎染，每天都有一些小雀跃，很快乐。当然，没有灵感、画不出东西的时候也会有些难过。兴奋的时刻倒确实有一个，我自学插画的时候上了水原希子的课，然后用她的方法画了些画发到Instagram上面，她给我点了赞，还发到了她的story里面。现在看来其实是一个很普通的行为，但当时我特别开心。 Q：除了插画之外，还有哪些事情能够激发你的热情？ A：我比较喜欢去尝试新的东西，比如去各地旅行，看不同地方的风景、当地人的生活。 Q：有哪些喜欢的艺术家？ A：我在做每个项目的时候都会去看不同艺术家的作品，最近在做版画，很喜欢David Hockney。之前做大猩猩动画的时候，我也有看过Frida的作品。也关注有些其他领域的艺术家，比如音乐方面，我很喜欢David Bowie。 Q：有什么配色方面的经验可以分享吗？ A：刚开始学画画的时候不知道怎么用色，发现自己只要颜色用得多就会不好看，于是就限制用色，每次只让自己用两三种颜色。我一般会用互补的颜色，比如蓝色和粉色，这样对比性强的色彩一下子就能出效果。熟练之后用色就自如起来了，再逐渐打破这个限制，跟着感觉走就好。再有，去看印象派的画，也是一个养成色感的方法，我会提炼出里面最主要的几种颜色的比例，用到自己的画里。 Q：未来有什么创作和发展计划？ A：对我来说，最理想的就是成立一个工作室，做独立插画师，跟志同道合的朋友一起工作。一定要是那种采光很好的工作室，有一面摆满绿植的墙，工作日我们几个朋友在一起工作，周末就做工作坊workshop。还要留一个房间做小型博物馆，里面摆满我们在世界各地买的书、各种各样好玩的东西，所有人都可以来参观。这样工作几年后，我还想再继续读书，学一个自己喜欢的专业，比如艺术教育。其实来英国留学后，发现很多国外的研究生同学年纪都不小，他们可能曾经从事着完全不同领域的工作，只因为喜欢画画就来到这里学习，从新开始。这是心态的问题，你完全可以不用按照这个世界认为的样子去活，可以做任何想做的事情，让自己的人生有更多可能性。 Q：期待看到行业发生哪些对创作者更友好的变化？ A：现在国内有各种各样的书展和艺术市集，给了插画师一个很好的平台去展示、售卖自己的作品。但参与进来的人都是熟悉的面孔，希望能有些新鲜的血液进来。其实，刚入行的插画师很需要帮助，比如商业合作怎样报价、怎样保护作品的版权等等，这样也能把更多的经历放在创作上。国外有经纪公司去帮你处理这方面的事务，也有像英国AOI这样的机构来制定一些有利于插画师的行业规则，希望国内这个行业的系统也能逐渐完善起来。
